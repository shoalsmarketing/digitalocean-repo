Frequently Asked Questions

How much time should I dedicate to thorough testing?

Meaningful evaluation requires several focused sessions spread across the trial period. Building actual funnels, testing automation, and processing transactions takes more time than feature browsing. Plan for hours rather than minutes if seeking confident conclusions.

Should I connect my real payment processor during trial?

Connecting actual payment processors enables realistic transaction testing. Test mode transactions verify flow without real charges. This connection carries forward if converting to paid subscription, making setup investment immediately valuable.

What happens to trial data after evaluation?

Created content, contacts, and configurations persist after trial. Converting to paid subscription restores full access to everything built. Declining subscription restricts feature access but preserves underlying data for future reconsideration.

Is support available during trial?

Support access functions during evaluation periods. Response times may differ from priority support available to higher-tier subscribers. Documentation and community resources supplement direct support for common questions.

Can I evaluate team features if working solo?

Team functionality testing requires additional user accounts. Creating test accounts for colleagues enables collaboration feature evaluation. Solo users can assess team capabilities through documentation and interface exploration.
